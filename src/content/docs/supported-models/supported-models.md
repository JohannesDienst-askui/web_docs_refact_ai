## Cloud Version of Refact

At the moment, Refact cloud supports the following models: 
- Refact LLM
- GPT 3.5 
- GPT 4 (Pro plan) 

## Self-Hosted Version of Refact

In Refact self-hosted you can select between the following models: 
- Refact LLM
- Starcoder         
- Wizard coder    
- Code Llama      
- Llama2
